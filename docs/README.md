        - [ ] ペナルティ解除ロジック (タイマー経過で移動速度パラメータを元に戻す)。
- [ ] **相打ち処理:**
    - [ ] 同時ヒット (両者の攻撃が同じフレームでお互いのハートボックスにヒット) の判定ロジック。
